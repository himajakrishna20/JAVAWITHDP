# ⚡ JAVAWITHDP

A compact collection of **Dynamic Programming problems in Java** focused on building strong problem-solving intuition for coding interviews.

## 🧩 What's Inside?

Core DP patterns implemented with clarity:

* Recursion → Memoization → Tabulation
* Knapsack variants (0/1 & Unbounded)
* Subsequence problems (LCS, LIS, Palindromes)
* Grid problems (Paths, Minimum/Falling Sum)
* Classic problems (House Robber, Frog Jump, Partition)


## 🎯 Purpose

> Learn DP the *right way* — by understanding patterns, not memorizing code.
